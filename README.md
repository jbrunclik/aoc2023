# Advent of Code 2023

My solutions for [Advent of Code 2023](https://adventofcode.com/2023) written in C#.

## Background

2023 was the year I joined the **IDNA (Identity and Network Access)** team at Microsoft. Since most of our components were written in C#, I used Advent of Code as a way to learn the basics of the language while having fun solving puzzles.

## Project Structure

```
Aoc2023/
├── Aoc2023.cs        # Main entry point with CLI
├── Solution.cs       # Base solution class
├── Day1/
│   ├── Day1.cs       # Solution implementation
│   └── input.txt     # Puzzle input
├── Day2/
│   └── ...
└── ...
```

## Requirements

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

## Running Solutions

Build and run a specific day's solution:

```bash
dotnet run -- <day>
```

For example, to run Day 5:

```bash
dotnet run -- 5
```
